## BulkLoader (Bot)
---

Telegram Bot to Bulk Downloading list of urls and Upload to Telegram.

### Features:

#### Upload list of urls (2 methods):
- send command `/link` and then send urls, separated by new line.
- send txt file (links), separated by new line.
